# Vertical Timeline of Prime Ministers

## Project Overview
This project is a **Vertical Timeline of Prime Ministers** built using frontend development technologies. It visually represents the chronological order of Prime Ministers along with their tenure and key highlights in a clean, scroll-based layout.

## Technologies Used
- HTML  
- CSS  
- (Optional: Tailwind CSS / JavaScript)

##  Features
- Vertical timeline layout  
- Chronological representation of Prime Ministers  
- Clean and responsive UI  
- Easy to understand and visually appealing design  

##  Project Structure
```
project-folder/
│── index.html
│── style.css
│── assets/
```

##  How to Run
1. Download or clone the repository  
2. Open `index.html` in any modern web browser  
3. Scroll to explore the timeline  

##  Purpose
This project was created to practice frontend development skills and present historical data in an interactive and visually engaging way.
